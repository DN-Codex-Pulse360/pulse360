# Walkthrough Rehearsal Checklist and Scoring Rubric (DAN-67)

## Rehearsal Checklist
- Presenter can complete DS-01/DS-02/DS-03 in `<= 15` minutes.
- Each scenario transition is explicit across Salesforce -> Databricks -> Data Cloud -> Agentforce.
- Scripted evidence references are shown with exact IDs and commands.
- Fallback wording is used correctly when a UI step is unavailable.
- No hardcoded score explanation is used; all metrics are traced to runtime outputs.
- Final summary links each scenario result to run ID evidence.

## Scoring Rubric (0-2 each, max 10)
| Dimension | 0 | 1 | 2 |
| --- | --- | --- | --- |
| Timing discipline | Exceeds 15 minutes significantly | Slightly over time | Completes in 15 minutes or less |
| Transition clarity | Layer changes unclear | Some transitions explicit | All transitions explicit and smooth |
| Evidence quality | Missing key proof | Partial proof and weak references | Full proof with exact dashboard/panel/runtime references |
| Fallback execution | No fallback handling | Fallback attempted but unclear | Fallback wording clear and non-blocking |
| Narrative confidence | Disconnected persona story | Partially coherent story | Clear persona narrative with measurable outcomes |

## Pass Threshold
- Minimum pass score: `8/10`
- Any score below `8` requires one remediation rehearsal before demo.
